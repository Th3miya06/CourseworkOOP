# Ticket Management System

This is a README file for setting up and running the Ticket Management System. The system consists of a Spring Boot backend and a React frontend.

## Prerequisites

Before running the system, ensure you have the following installed:

- Java Development Kit (JDK) 8 or above
- Node.js and npm (Node Package Manager)



1. Navigate to the backend directory:
open the TicketingSystem_w2084732 folder with IntelliJ 

2.open vs code and open the fronted GUI folder TicketFrontend in vs code

3.open the terminal in frontend and type cd ticketing-GUI
make sure you have react js installed

4.if it is in the respective directory type npm run dev

5.the froentend Gui now should run at  http://localhost:5173/ this is connecting to the backend via endpoints in the ticketing controller at the backend 

6.backend url is http://localhost:8080/

7. Run the springboot application and enter the configurations . By communicating through the websocket the real time data will be displayed.

CLI is the folder named as JavaCwTesting(This is the CLI implementation) open this in intellij
